# 🎬 CineCatálogo

Aplicação Web desenvolvida em **React** para o trabalho final da disciplina de Front-End. O sistema permite ao usuário navegar por um catálogo de filmes e séries, pesquisar títulos, visualizar informações detalhadas, adicionar favoritos e receber recomendações aleatórias através de uma roleta.

# 📖 Descrição do Projeto

O **CineCatálogo** é uma Single Page Application (SPA) desenvolvida utilizando React e React Router DOM.

A aplicação simula um catálogo de filmes e séries, oferecendo uma interface moderna, intuitiva e totalmente navegável sem recarregar a página.

Durante o desenvolvimento foram utilizados conceitos de:

* Componentização;
* Props;
* Hooks (useState);
* React Router DOM;
* CSS Modules;
* Manipulação de listas;
* Filtros de pesquisa;
* Navegação SPA.

---

# 🎯 Objetivo

Desenvolver uma aplicação Front-End utilizando React que permitisse ao usuário:

* visualizar filmes e séries;
* pesquisar títulos;
* acessar informações detalhadas;
* adicionar e remover favoritos;
* navegar entre páginas utilizando React Router.

# ▶️ Como executar o projeto

## Clone o repositório

```bash
git clone <url-do-repositorio>
```

## Entre na pasta

```bash
cd catalogo-filmes-series
```

## Instale as dependências

```bash
npm install
```

## Execute o projeto

```bash
npm run dev
```

O projeto ficará disponível em:

```
http://localhost:5173
```
